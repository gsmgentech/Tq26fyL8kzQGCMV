Remote Support — USB Redirect

Customer Instructions for Remote Service

Overview: You will use Radmin VPN to create a secure connection and USB Redirector (Customer Module) to share your USB device with the technician. Administrator rights are required to complete the setup.

Step-by-Step Guide

  1. Download the required installers (links are available at the bottom of this page).
  2. Install Radmin VPN:
    • Run the installer as Administrator.
    • Allow driver installation when prompted.
    • Open Radmin VPN and click the Power (On) button to enable it.
    • Click Network → Join Network from the top menu.
    • Enter the Network Name and Password provided by the technician.
    • Confirm to join. Once connected, the VPN should show you are online.
  3. Install USB Redirector Customer Module (run as Administrator).
    • Enter the Remote technician address given by the technician.
    • Plug in the USB device to share. If already plugged in, re-plug it to continue.
    • Once shared, the technician will be able to connect to it.
  4. When the session is complete, stop sharing the device and close the software.

Troubleshooting

  • No VPN IP Address: Restart Radmin VPN and rejoin the network.
  • USB Device Not Detected: Ensure USB Redirector is running as Administrator.
  • Installer Blocked by Antivirus: Confirm the installer source and whitelist if verified safe.
