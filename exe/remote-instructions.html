Remote Support — USB Redirect

Remote bypass / remote service instructions (customer consent required)

Overview: We use Radmin VPN to create a secure virtual LAN and USB Redirector (customer module) to forward the customer’s USB device to the technician’s computer. Admin rights are required.

Step-by-step

  1. Get explicit consent from the customer and log purpose + time.
  2. Download the installers using the links on the right.
  3. Install Radmin VPN (run as Administrator, allow drivers).
  4. Join the VPN network using the technician’s name & password.
  5. Install USB Redirector (run as Administrator, share device).
  6. Technician connects through VPN to access the shared device.
  7. End session: unshare device and confirm with customer.

Troubleshooting

  • VPN IP missing — restart Radmin VPN and rejoin the network.
  • USB not visible — run USB Redirector as Administrator.
  • Antivirus flags installer — verify source and whitelist if safe.
